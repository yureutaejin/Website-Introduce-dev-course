About Page

한국의 수 많은 학생들이 개발분야에 입문하지만 본인의 진로에 관해 깊게 생각해보지 않은 채로, 학교나 학원의 커리큘럼을 따라서 공부합니다.

단순한 웹개발자 양산을 막기 위해서 초보 개발자를 위한 간단한 커리큘럼 및 코스를 정리하는 사이트를 만들어보았습니다.

이 사이트를 통해 개발분야에 어떤 진로들이 있으며, 어떤 것을 공부해야하는지, 어떤 툴이나 라이브러리, 프레임워크가 있는지, 도움이 되는 사이트는 무엇이 있는지 간단하게라도 알아갔으면 좋겠습니다.

해당 웹사이트는 프론트는 node js express, 백엔드는 flask 환경을 기준으로 개발되었습니다.

front는 기초 css, html, vanilla js, nodejs를 이용해서 개발했습니다. webpack 등으로 구조를 간단화할 예정입니다.

backend는 flask로 front 요청에 응답하는 구조로 만들었습니다. 사이트의 지속적 유지보수를 위해 만들었으며, CRUD와 REST API 기능을 제대로 추가할 것입니다.

정확히 동일하게 구현을 원하는 경우에는 git readme 지시사항에 따라 Dockerfile을 통해서 실험환경을 구성해주세요.
